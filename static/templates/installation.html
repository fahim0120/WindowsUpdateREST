Study to Measure Daily Computer Power Usage



Software Installation Manual




Step 1: Download the Software. It is a zip file, named: “PowerMeterApp.zip”.  

Step 2: Extract (right mouse button click and extract) the zip file to get the executable file named: “PowerMeterApp.msi”  

Step 3: Double click on the executable file “PowerMeterApp.msi”.  

Step 4: Click on the Next button.  
 

Step 5: Click on the Next button.  
 

Step 6: Click on the Install button.  
 

Step 7: Click on the Yes button.
(Depending on your download location, you will see a different directory on the red boxed area.)  
 

Step 8: Click on the Finish button.
Leave the “Launch PowerMeterApp” check box checked.  
 

Step 9: Enter your MTurk ID and Windows Operating System Version. Submit.  
 

Step 10: You will get the 5-letter MTurk completion code. Enter the code in MTurk for compensation first.
Then, click on the OK button.
